- Build on top of TCP

# Request
Has the following schema:

```
request={
    host:"vinayaklokhande.com",
    port:"80",
    headers:{
        content-type:"JSON/plain-text",
        content-length:51
    },
    path:"/names",
    method:"GET|POST|PUT",
    body:"some data to be sent or json data"
}
```
# Response
Has the following schema:
```
respose={
    statuscode:200|400|500
    headers:{
        content-type:"JSON/plain-text",
        access-control-allow-origin:"somedomain.com"
    },
    body:"some data to be sent or json data"
}
```
